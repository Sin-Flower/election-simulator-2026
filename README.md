# 제9회 전국동시지방선거 시·도지사 선거 시뮬레이터

전국 3,558개 읍·면·동의 실제 선거인수·후보 득표(관내사전·당일·관외사전·거소 포함)를 바탕으로 한
몬테카를로 선거 시뮬레이터입니다. (광주·전남은 전남광주통합특별시로 합산)

데이터 출처: 중앙선거관리위원회 선거통계시스템 (info.nec.go.kr) · 제9회 전국동시지방선거(2026-06-03) 개표단위별 개표결과.

## 배포 (GitHub Pages)
1. 이 저장소를 GitHub에 올립니다.
2. Settings → Pages → Branch를 `main` / `/ (root)`으로 설정.
3. 몇 분 뒤 `https://<사용자명>.github.io/<저장소명>/` 에서 열립니다.

순수 정적 단일 페이지(`index.html`)라 별도 빌드가 필요 없습니다.
